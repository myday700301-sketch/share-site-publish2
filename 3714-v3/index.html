基礎定位
  • 台股中型上市光電整合股，屬於中型高β事件驅動鎖價型趨勢股。

核心主線
  • A 級核心主線：車用光電：車外 / 車內照明、ADB、Mini / Micro LED 車用顯示。
  • A / B 級交會主線：先進顯示：Mini LED、Micro LED、透明顯示、HUD 與背光升級。
  • 題材重評主線：智能感測：IR、VCSEL、感測模組、安防 / 穿戴 / 工業感測。
  • 中長線補充主線：AI 光通訊：Micro LED、VCSEL、Tx/Rx、CW-DFB LD、矽光子 / CPO / 外置光源。

比較籃子
  • 主比較籃子維持光電/感測；20台股peer與TAIEX僅作相對強弱校正。

題材分級
  • A 級：真正改變產品 mix、毛利率、客戶層級或量產節奏的高值應用
  • B 級：可帶來短中期營收改善，但估值彈性不如 A 級
  • C 級：族群輪動、媒體敘事、低成長或低毛利舊業務，不能取代 A 級主線
