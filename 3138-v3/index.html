基礎定位
  • 台股中小型射頻天線與衛星通訊股，主體在 RF 元件 / 天線模組 / 測試驗證，估值彈性來自低軌衛星與無人機。

  • 市場定位：通信網路
  • 非屬類型：不是純低軌衛星終端量產股，需區分天線、設備、測試服務與衛星商轉進度。
  • 常見切換情境：衛星題材點火、急漲後量縮、驗證公告再攻、高檔洗盤。

核心主線
  • A 級核心主線：RF 元件、天線模組與量測 / 認證服務，屬營收主體。
  • A / B 級交會主線：低軌衛星 Ku / Ka 頻段終端與地面接收站，屬高值主線。
  • 題材重評主線：無人機、車聯網與工業 / 生醫無線應用，屬題材擴張主線。
  • 中長線補充主線：MWC、法說、衛星驗證與國際客戶談判，屬事件 Beta 核心。

比較籃子
  • RF / 衛星：6285啟碁、3596智易（只看網通溫度）、3443創意（只看衛星題材溫度）、3152璟德

題材分級
  • A 級：真正改變產品 mix、毛利率、客戶層級或量產節奏的高值應用
  • B 級：可帶來短中期營收改善，但估值彈性不如 A 級
  • C 級：族群輪動、媒體敘事、低成長或低毛利舊業務，不能取代 A 級主線
