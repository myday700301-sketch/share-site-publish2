固定設定卡

固定設定

基礎定位
  • 台股上市汽車工業精密金屬加工 / 車用零組件股，營收主體在車用精密零件，並以醫療、工業應用與減速機產品提供估值彈性。
  • 股性主型偏「中型精密金屬加工股或事件驅動股，需分開看車用主體營收、產品 mix 升級與減速機題材重評」
  • 市場定位：汽車精密零組件
  • 非屬類型：不是純機器人概念股，也不是單一自行車題材股；營收主體仍是汽車精密零組件與金屬加工能力，減速機與新應用屬重評選擇權。
  • 常見切換情境：車用新品放量、產品 mix 上修、減速機 / 自動化題材重評、急漲後回測事件成本再攻、法說或月營收驅動趨勢延伸。

核心主線
  • A 級核心主線：汽車引擎、轉向、傳動、煞車與底盤等精密零件，屬營收主體。
  • A / B 級交會主線：醫療器械、工業通訊 / 自動化與高階自行車精密件，屬產品 mix 擴張主線。
  • 題材重評主線：TUF ONE 精密行星減速機、AGV / 機械手臂 / 智慧製造應用，屬估值重評主線。
  • 中長線補充主線：EV、氫燃料電池、新廠 / 新客戶 / 第一層 驗證與量產進度，屬中長線事件 Beta。

比較籃子
  • 汽車精密零件 / 傳動減速機：4551智伸科、2228劍麟、1536和大、4571鈞興-KY、4583台灣精銳

題材分級
  • A 級：真正改變產品 mix、毛利率、客戶層級或量產節奏的高值應用
  • B 級：可帶來短中期營收改善，但估值彈性不如 A 級
  • C 級：族群輪動、媒體敘事、低成長或低毛利舊業務，不能取代 A 級主線
