基礎定位
  • 大型權值股的事件再定價與 AVWAP 均值回歸型波段標的；短線受高振幅與高當沖影響
  • 股性主型偏「設備 / 系統整合股或平台型中大型股，需區分設備接單、產業資本支出與權值性」
  • 市場定位：電源管理 / 散熱 / 電動車與資料中心
  • 非屬類型：不得以高振幅替代beta；未收復事件 AVWAP 前以反彈壓力管理，同族群 籃子僅自身不能用同業估值升信心
  • 常見切換情境：成本帶守穩→1875→1912至1943→2055/2108分段升級；任一階段失敗即退回前一級

核心主線
  • A 級核心主線：其他電子主體業務與核心產品，屬營收主線
  • A / B 級交會主線：高值產品 / 新應用 / 新客戶，屬估值彈性主線
  • 題材重評主線：產業景氣、報價、接單或產能變化，屬基本面主線
  • 中長線補充主線：法說、月營收、重大訊息、展會與客戶驗證，屬事件 Beta 核心

比較籃子
  • 加權指數相對強弱可用但波動係數／相關係數不可用；同族群籃子僅自身，美股代理標的無序列

題材分級
  • A 級：真正改變產品 mix、毛利率、客戶層級或量產節奏的高值應用
  • B 級：可帶來短中期營收改善，但估值彈性不如 A 級
  • C 級：族群輪動、媒體敘事、低成長或低毛利舊業務，不能取代 A 級主線

常見劇本
  • 成本帶守穩→1875→1912至1943→2055
  • 2108分段升級；任一階段失敗即退回前一級

固定校正原則
  • A 級主線權重高於 C 級敘事
  • 題材驅動、基本面修復、純籌碼推動必須分開判讀
  • 分 K 只能補充波段背景，不可主導波段結論
